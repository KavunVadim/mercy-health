# Design System — Благодійний фонд «Милосердя та здоров'я»

> Референс: savelife.in.ua — структура та функціональність.
> Адаптація: Modern Professional Humanitarian (чистий, високий контраст, преміальність).

---

## Філософія Дизайну
Сайт має виглядати як сучасна, прозора та надійна організація. Ми використовуємо принципи Apple (великі заголовки, багато простору, розмиття фону) у поєднанні з активними закликами до дії (CTA) у стилі Savelife.

## Кольорова палітра (Professional Palette)

```css
/* === ОСНОВНІ ФОНИ === */
--bg-primary: #ffffff;
--bg-secondary: #f5f5f7; /* Apple off-white — для чергування секцій */
--bg-footer: #f5f5f7;

/* === БРЕНДОВІ КОЛЬОРИ === */
--color-accent: #1ea2fe; /* Logo Blue — для навігації, іконок, акцентів */
--color-donate: #e8401c; /* Donate Red — виключно для кнопок пожертви */

/* === ТИПОГРАФІКА === */
--color-text-primary: #1d1d1f;   /* Глибокий чорний для заголовків */
--color-text-secondary: #424245; /* Темно-сірий для основного тексту */
--color-text-muted: #86868b;    /* Світло-сірий для другорядних деталей */

/* === UI ЕЛЕМЕНТИ === */
--radius-standard: 4px; /* Для всіх кнопок та карток (строгий, професійний вигляд) */
--border-light: rgba(0, 0, 0, 0.05);
```

---

## Типографіка
- **Заголовки**: Oswald. Використовувати великий кегль (4.5rem+ для Hero), uppercase, жирне накреслення.
- **Текст**: Inter. Сучасний, максимально читабельний шрифт. Основний розмір збільшено для кращого фокусування.

## Кнопки
- Всі кнопки мають однаковий радіус закруглення (4px).
- **Primary (Donate)**: Червоний фон, білий текст. Використовується для прямих закликів до пожертви.
- **Secondary**: Блакитний фон або контурна кнопка. Для ознайомлення з проектами.

---

## Структура та Конверсія
1. **Hero**: Захоплення уваги великим заголовком та чітким описом місії.
2. **Stats**: Пряме підтвердження діяльності через цифри.
3. **Projects**: Картки проектів з прогрес-барами для наочності потреби.
4. **CTA**: Постійна наявність кнопки «Допомогти» у шапці та після кожної великої секції.
